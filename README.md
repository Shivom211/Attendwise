# AttendWise

AttendWise is a Python-based CLI application that helps students track attendance
and make informed decisions about whether they can safely skip classes.

## Problem Statement
Many students struggle to keep track of subject-wise attendance and often skip
classes without knowing the long-term impact. AttendWise solves this by providing
clear attendance analytics and skip-decision support.

## Features
- Add and manage subjects
- Update attendance after each class
- View subject-wise and overall attendance percentage
- Decide whether the next class can be skipped safely
- Configurable minimum attendance requirement
- Simple and human-friendly CLI interface

## Technologies Used
- Python 3
- JSON (for persistent storage)
- Command Line Interface (CLI)
