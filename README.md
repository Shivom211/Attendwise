# 🎓 AttendWise

An intelligent attendance management system built using **Python, Flask, SQLite, HTML, CSS, and JavaScript**. AttendWise helps students track attendance, set attendance targets, and make smarter decisions about skipping classes.

---

## 🚀 Features

- 👤 User Registration & Login
- 📚 Subject Management
- 📊 Attendance Tracking
- ⭐ Intelligent Skip Decision
- 📈 Overall Attendance Dashboard
- 📉 Attendance Graph
- 🎯 Custom Attendance Target
- 🤖 ML-based Attendance Prediction
- 💾 SQLite Database
- 📄 JSON Support

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Shivom211/AttendWise.git
```

2. Go to the project folder

```bash
cd AttendWise
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
